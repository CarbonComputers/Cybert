# Changelog — Cybert

All notable changes will be documented here.

---

## [3.0.0] — 2025
### Added
- Initial public release of Cybert v3.0 BETA
- Updated PCB with M.2 and USB 3.0 A 
- Enhanced battery management  
- Improved keyboard matrix firmware  

### Fixed
- Display shift issue  
- Power stability with CM5  
- Improved power button
- Improved keyboard drop switch function

---

## [2.0.0] — 2024
### Added
- BMS Battery indicator 
- Touchpad for tracking
- left and right click
- Stereo audio

### Fixed
- Slim Display issue

---

## [1.0.0] — 2024
Initial prototype release
- Used a thicker HDMI display, slim display did not work
- No touchpad or trackpad 
- No audio
