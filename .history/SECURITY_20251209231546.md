## Reporting Security Issues

If you discover a vulnerability or security risk in the Cybert hardware, firmware, or software:

**Do not open a public GitHub issue.**

Instead, report privately to:
**security@carboncomputers.us**

We aim to acknowledge reports within 72 hours and provide updates throughout the resolution process.

## Supported Versions

| Version | Status |
|--------|--------|
| v3.0   | Supported |
| v2.x   | Partial Support |
| v1.x   | Legacy |

## What to Report
- Firmware vulnerabilities  
- Power/battery safety issues  
- USB or interface-level security concerns  
- OS-level security risks unique to Cybert  

Thank you for helping keep users safe.
